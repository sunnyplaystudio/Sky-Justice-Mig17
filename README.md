# MIG-17: Sky Justice 🚀

MIG-17 Sky Justice is a high-speed political satire arcade game where you pilot a legendary fighter jet against various global targets.

## 🕹️ [PLAY THE GAME NOW!](BURAYA_GITHUB_PAGES_LINKINI_YAPISTIR)

## 📖 About the Game
Experience retro arcade mechanics combined with modern political satire. Dodge, shoot, and aim for the high score while navigating the stormy skies.

### Features:
- **Fast-paced Gameplay:** Classic arcade shooter mechanics.
- **Satirical Elements:** Features various global figures as targets.
- **Dynamic Difficulty:** The game gets faster as your score increases.
- **High Score Tracking:** Save your best runs locally.

## 🎮 How to Play
- **Desktop:** Use your **Mouse** to steer the jet and **Click** to fire missiles.
- **Mobile:** **Slide** your finger to move and **Tap** to shoot.
- **Pause:** Use the Pause button on the top left to take a break.

## 🛠️ Technical Details
- Built with HTML5 Canvas and Pure JavaScript.
- Mobile-optimized and responsive design.
- Chrome Auto-Translate protection implemented.
